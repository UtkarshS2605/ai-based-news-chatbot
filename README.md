# 🧠 AI News Chatbot

An AI-powered full-stack news chatbot built using **React, FastAPI, OpenAI, and NewsAPI** that fetches real-time news and generates intelligent AI summaries.

⚙️ Installation & Setup
1️⃣ Clone Repository
git clone https://github.com/UtkarshS2605/ai-based-news-chatbot.git
2️⃣ Backend Setup
cd ai-based-news-chatbot

Create virtual environment:

python -m venv .venv

Activate virtual environment:

Windows
.venv\Scripts\activate

Install dependencies:

pip install -r requirements.txt
3️⃣ Create .env File
NEWS_API_KEY=your_newsapi_key
OPENAI_API_KEY=your_openai_key
4️⃣ Run Backend
uvicorn app.main:app --reload

Backend runs on:

http://127.0.0.1:8000
5️⃣ Frontend Setup
cd frontend
npm install
npm run dev

Frontend runs on:

http://localhost:5173
